**This is a sample golang application.**

**To start run:**
go run server.go

**Supported API's**
- [x] GET /              - Home page.
- [x] GET /articles      - Returns all articles.
- [x] GET /articles/{id} - Get article by id.
- [x] POST /article      - Create new article.
- [x] DELETE /article/id - Delete article by id.
